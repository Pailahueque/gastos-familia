¿Por qué se activó el Modo Local?

Es probable que aún falte configurar las Reglas de Seguridad en tu consola de Firebase. Sigue estos simples pasos para solucionarlo:

  1. Entra a la Consola de Firebase.
  2. Ve a tu proyecto y haz clic en Firestore Database en el menú izquierdo.
  3. Entra en la pestaña superior llamada Reglas (Rules).
  4. Reemplaza el código por este bloque completo:
    5. rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}
  5. Haz clic en el botón azul de Publicar. ¡La app se sincronizará automáticamente!
